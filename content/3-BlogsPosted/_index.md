This section lists the knowledge sharing blog posts published on [AWS Study Group VN](https://www.facebook.com/groups/awsstudygroupfcj):

### [Blog 1 - Amazon EKS Supports Control Plane Egress Routing Through VPC](3.1-Blog1/)
This post introduces the Customer-routed control plane egress feature in Amazon EKS, enabling businesses to actively route and control all outbound traffic of the Kubernetes Control Plane through their own VPCs.

### [Blog 2 - Amazon EKS Auto Mode and Istio Ambient Mesh Integration](3.2-Blog2/)
This post analyzes the combination of Amazon EKS Auto Mode, compute automation via Karpenter and Bottlerocket, and Istio Ambient Mesh, a sidecarless service mesh with ztunnel and waypoint proxies.

### [Blog 3 - AWS Continuum - Intelligent Vulnerability Management Powered by AI](3.3-Blog3/)
This post introduces AWS Continuum, a security initiative from AWS that uses Artificial Intelligence (AI) to optimize and automate the vulnerability management lifecycle for enterprises.
